# AI图像处理器 (AI Image Processor)

这是一个基于Django框架的AI图像处理应用程序，专门用于机械零件图纸的智能分析。该系统结合了AI技术与RAG（检索增强生成）功能，能够自动识别和分析机械零件图中的各种制造特征。

## 功能特性

- **AI图像分析**：使用Kimi服务对上传的机械零件图进行智能分析
- **特征识别**：自动检测零件图中的槽、孔、倒角、台阶等各种制造特征
- **RAG系统**：支持特征数据的标注、存储和检索
- **多用户支持**：区分普通用户上传和管理员标注功能
- **数据管理**：提供历史记录查看和数据管理功能

## 技术栈

- **后端**: Django框架
- **前端**: Bootstrap + jQuery
- **AI服务**: Kimi API（用于图像分析）
- **数据库**: Django ORM（支持多种数据库）
- **文件存储**: 支持图像文件上传和管理

## 主要模块

### 1. 图像处理模块
- 支持图像上传和预处理
- 与Kimi服务集成进行AI分析
- 结果展示和历史记录管理

### 2. RAG系统
- 特征标注界面（管理员功能）
- 特征数据存储（槽、孔、倒角、台阶等）
- 特征表格生成（包含特征、操作、前置操作、机床、工具等信息）

### 3. 用户管理
- 普通用户上传分析功能
- 管理员标注和审核功能
- 数据审核流程

## 安装部署

### 环境要求
- Python 3.8+
- Django框架
- 图像处理相关依赖库

### 安装步骤
1. 克隆项目到本地
2. 创建虚拟环境并安装依赖
   ```bash
   python -m venv ai_web
   ai_web\Scripts\activate  # Windows
   pip install -r requirements.txt
   ```
3. 初始化数据库
   ```bash
   python manage.py migrate
   ```
4. 创建超级用户（可选）
   ```bash
   python manage.py createsuperuser
   ```
5. 启动开发服务器
   ```bash
   python manage.py runserver
   ```

## 使用说明

### 普通用户功能
1. 访问主页面上传机械零件图
2. 系统自动调用AI服务进行分析
3. 查看分析结果和历史记录

### 管理员功能
1. 登录管理员界面
2. 进入RAG标注页面
3. 对上传的图像进行特征标注
4. 审核和管理标注数据

## 配置说明

### AI服务配置
- 需要在`settings.py`中配置Kimi API密钥
- 确保网络可以访问外部AI服务

### 文件上传配置
- 默认上传路径：`uploads/%Y/%m/%d/`
- 支持常见图像格式（JPG, PNG等）

## 注意事项

1. 确保AI服务API密钥正确配置
2. 定期备份数据库中的标注数据
3. 监控文件存储空间使用情况
4. 生产环境中需配置适当的静态文件服务

## 维护和扩展

- 可以通过修改`models.py`扩展数据模型
- 支持添加新的AI分析服务
- 可以定制化特征识别算法
- 支持多语言界面扩展

## 故障排除

如果遇到问题，请检查：
1. 虚拟环境是否激活
2. 依赖包是否完整安装
3. 数据库连接是否正常
4. AI服务API是否可用
5. 文件权限是否正确设置

如有其他问题，请查看Django日志文件或联系技术支持。
